Testing everything is working

After installation has completed a new lambda function will have been registered with the supplied Stack Name by default this will be AWS-Events2Panther.

Simulating an EC2 instance state change

Find your lambda functions, if you imported with default values it will be named AWS-Events2Panther and should be listed on aws.amazon.com/lambda/home

Sample test data

{
   "id":"7bf73129-1428-4cd3-a780-95db273d1602",

   "source":"aws.ec2",
   "account":"123456789012",
   "time":"2019-11-11T21\:29\:54Z",
   "region":"us-east-1",
   "resources":[
